# SVN Web Browser (SVN 网页浏览器)

[English](./README.md) | **中文**

## 简介
这是一个轻量级、无刷新、拥有现代 UI 的 SVN 网页端浏览工具。
通过极其简单的 PHP 后端驱动环境下的 `svn` 命令行进行安全交互，再由原生的 HTML/CSS/JS 提供丝滑的前台视窗体验。不需要繁重框架，只要服务器有 PHP 和 SVN，即可做到**即插即用**。

## 🌟 功能特性
- **纯粹极简**：零第三方 NPM 包，零复杂的 Node 服务器，无数据库要求。
- **动态树状游览**：输入账号连接后，全异步加载！可在页面任意点击目录进入，并通过“面包屑”即时点击跳回上一级。
- **内侧文件阅读**：自带纯文本代码的预览视图。遇到想看的纯代码或文档，点击一行直接覆盖在当前页内预览（调用 `svn cat` 操作），无需离开页面。
- **完整的提交信息**：精准利用 XML 取值，能正确映射并美化输出 文件类型（目录或文件）、容量大小与详细的 SVN 修改号、作者以及日期。
- **优雅的状态捕捉**：无论你是输错了密码、写错了SVN地址、亦或者服务器卡顿，都能弹出明确详尽的中文/原生报错窗口帮助排查。

## 🚀 部署与使用

### 环境要求
1. 系统的全局终端必须支持 `svn`（可通过 `svn --version` 测试）。
2. Web 服务器支持 PHP 解析（例如 Apache、Nginx 的 php-fpm 或者 MAMP 架构等）。

### 使用步骤
1. 将本文件夹（如 `svnWeb`）移动至任何能够解析 PHP 的 Web 目录下。
2. 浏览器打开入口地址。
   >*如果仅仅想本地测试，可以在项目根目录开启内建服务器：*
   >`php -S 0.0.0.0:8888` 
   >随后访问 `http://127.0.0.1:8888` 即可。
3. 在左侧菜单栏输入你想连接的 `svn://...` 或者 HTTP 协议的具体链接及**对应账号密码**，点击**连接并浏览**即可起飞。

## 📦 目录结构

```text
svnWeb/
├── api.php             # 核心 API 中转：验证拦截、跑 svn CLI 并在 XML 与 JSON 互切
├── index.html          # 主视图（毛玻璃右侧菜单、面包屑与主体表格和预览区）
├── css/
│   └── style.css       # 原生现代极简响应式样式与动画 CSS
└── js/
    └── app.js          # 全部的前端生命周期、按钮防抖、Fetch 路由控制引擎
```

## ⚠️ 注意事项
* **命令缓存**：由于我们使用的底层命令附带了 `--no-auth-cache` 参数，你在网页里的连接验证不会去污染或覆盖系统原有的凭证！
* **二进制文件**：对于图片或可执行文件等特殊文件，网页的纯文本预览试图可能会出现乱码行为。
* **安全性**：由于工具具备通过 PHP 执行 `proc_open` 拼接 Shell 指令的权限，若是放在外网环境部署需保证严格的安全策略配置，避免恶意拼接利用。
